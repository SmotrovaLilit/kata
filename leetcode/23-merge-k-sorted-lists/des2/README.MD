# Merge lists one by one

## Time complexity : 
O(kN) 
- We can merge two sorted linked list in O(N) time where N is the total number of nodes in two lists.
-  k is the number of linked lists.
## Space complexity : 
O(1)

We can merge two sorted linked list in O(1) space.
